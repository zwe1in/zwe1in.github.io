const

1.const作用

含义：const声明的变量，它的值不能再被改变

  • 声明方式：
  • 作用：
    1. 类型检查；const常量具有类型，编译器可以进行安全检查；#define宏定义没有数据类型，只是简单的字符串替换，不能进行安全检查。
    2. 防止修改；起保护变量值的作用。
    3. 节省空间；const定义只给了地址，#define是立即数；const在程序运行过程只有一份拷贝，#define有若干个拷贝。

2.const常量

2.1 常量定义

2.2 常量的跨文件访问

非常量在跨文件使用时，只需要在应用的文件处指定extern。

常量在跨文件使用时，需要在所有涉及到它的文件处都显式地指定extern const，并且在其中一处对它初始化。

3.const与指针

const与指针的三种结合：

3.1 指向常量的指针

  • 可以指向变量。
  • 指向的对象的值不能改变

3.2 常指针

  • 常指针必须初始化，指针的指向不能修改，但是指向元素的值可改。

！注意，把一个常量的地址赋值给常指针ptr时，由于ptr指向一个变量且不可改变指向，导致会报错。

解决上述问题，可以改为指向常量的常指针(const int *const ptr)或者指向常量的指针(const int *ptr)。

3.3 指向常量的常指针

指向目标需要为常量，指针指向不变，需要初始化。

4.const与函数

4.1 const 参数

(1) 传递过来的参数及指针本身在函数内不可变，无意义！

(2) 在地址传递的方式中，保护输入不变，同时提高效率避免复制传参。

(3) 使用常引用，提高效率同时防止修改，类似于(2)

函数内部需要用到对象a但不能改变a，复制副本会浪费时间，为了提高效率，可以通过引用提高效率，同时用const避免对象被修改。
！！注意在普通类型void func(int a)之下，就没必要这样做了，因为普通类型复制、析构效率十分快。

4.2 const 返回值

5.const与类

5.1 const 成员变量

不可修改，而且只有const成员函数可以访问。

5.2 const 成员函数

对所有的成员变量都不可以修改，可以访问const成员变量。
